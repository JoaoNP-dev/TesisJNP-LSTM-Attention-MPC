# TesisJNP-LSTM-Attention-MPC
# Sistema de Gestión Energética basado en LSTM + MPC

Este repositorio contiene la implementación del modelo desarrollado en la tesis:

"Diseño de un controlador predictivo basado en redes LSTM con mecanismo de atención para la gestión eficiente de la demanda energética en una vivienda autosostenible"

## Descripción

El sistema integra:
- Modelo predictivo LSTM con mecanismo de atención
- Controlador MPC (Model Predictive Control)
- Modelado energético de vivienda autosostenible

## Componentes principales

- Generación de dataset maestro (clima, ocupación, demanda)
- Modelo de predicción de demanda
- Controlador MPC
- Simulación de escenarios energéticos

## Objetivo

Optimizar la gestión energética priorizando:
- Continuidad del suministro (ENS)
- Confort del usuario
- Uso eficiente de energía solar

## Reproducibilidad

El código está organizado para permitir la replicación de los resultados obtenidos en la tesis.
